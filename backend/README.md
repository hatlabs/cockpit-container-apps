# Cockpit Container Apps - Backend

Python backend for the Cockpit Container Apps module.

## Development

See the [main README](../README.md) for development instructions.

## Structure

- `cockpit_container_apps/` - Main package
  - `cli.py` - CLI entry point
  - `commands/` - Command handlers
  - `vendor/` - Vendored utilities from cockpit-apt
- `tests/` - Test files
- `pyproject.toml` - Python package configuration
